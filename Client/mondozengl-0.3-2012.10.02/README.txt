RU:
Положите каталог MondoZenGL рядом с каталогами src, lib и headers от ZenGL. Для компиляции с Direct3D версией ZenGL не забудьте указать путь к каталогу src/Direct3D.

EN:
Put directory MondoZenGL near directories src, lib and headers of ZenGL. For compilation with Direct3D version of ZenGL don't forget to set a path to directory src/Direct3D.
